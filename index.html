Biljeske s predavanja

1. Lekcija

CSS (Cascading Style Sheets) je jezik koji se koristi za oblikovanje web stranica.
Njime mijenjamo izgled HTML elemenata – boje, fontove, veličine, raspored i pozicije na stranici.

Primjer Inline CSS-a:

<p style="color:red;">Tekst</p>

Primjer Internal CSS-a:

<style>
p { color: blue; font-size: 18px; }
</style>

Primjer External CSS-a:

<link rel="stylesheet" href="stil.css">

2. Lekcija

CSS box model je osnovni koncept u CSS-u koji opisuje kako se elementi prikazuju i kako zauzimaju prostor na stranici.
Svaki HTML element se smatra pravokutnom kutijom (boxom) koji se sastoji od 4 glavna dela:

Content (sadržaj) – prostor gdje se nalazi tekst, slika ili drugi sadržaj elementa.

Padding – prostor između sadržaja i ivice (bordera) elementa. "Gura" sadržaj ka unutra.

Border – linija koja okružuje padding i sadržaj.

Margin – prostor izvan border-a, tj. razmak između ovog elementa i drugih elemenata oko njega.
