Biljeske s predavanja

CSS (Cascading Style Sheets) je jezik koji se koristi za oblikovanje web stranica.
Njime mijenjamo izgled HTML elemenata – boje, fontove, veličine, raspored i pozicije na stranici.

Primjer Inline CSS-a:

<p style="color:red;">Tekst</p>

Primjer Internal CSS-a:

<style>
p { color: blue; font-size: 18px; }
</style>

Primjer External CSS-a:

<link rel="stylesheet" href="stil.css">
